Overview

Ohouse Visual Asset Library의 기본 체계와 운영 원칙을 정리한 문서입니다. 제품 전반에서 사용되는 시각 자산들이 일관된 톤으로 읽히도록 돕는 가이드라인을 제공합니다.

본문 작성 예정 Notion의 Visual System 페이지 콘텐츠가 여기에 텍스트 형태로 정리됩니다.

구성

Library는 세 갈래로 정리됩니다.

  • Visual Language — Authenticity를 잇는 네 가지 조형 원칙(Softness · Dream · Clarity · Stability)
  • Assets — Icon · 2D · 3D · Motion 네 가지 포맷의 그래픽 자산
  • 이 페이지(Overview) — 위 둘을 어떻게 함께 운영하는지에 대한 안내
